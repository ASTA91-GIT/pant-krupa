# 🌿 Pant Krupa Farmhouse

> **Escape the city. Experience nature. Create unforgettable memories.**

Pant Krupa Farmhouse is a modern, responsive website built to showcase the beauty and amenities of Pant Krupa Farmhouse while providing visitors with a seamless booking experience.

Designed with simplicity and elegance, the website allows guests to explore the property, view facilities, browse images, check location details, and submit booking inquiries through an intuitive interface.

---

## ✨ Features

* 🏡 Beautiful landing page
* 📱 Fully responsive design
* 📸 Farmhouse gallery
* 🛏️ Amenities section
* 📍 Integrated Google Maps location
* 📝 Online booking/inquiry form
* 📞 Contact information
* ⚡ Fast loading and optimized performance
* 🎨 Clean and modern UI

## 🚀 Getting Started

Clone the repository

```bash
git clone https://github.com/ASTA91-GIT/pant-krupa.git
```

Move into the project

```bash
cd pant-krupa
```

Install dependencies

```bash
npm install
```

Run the development server

```bash
npm run dev
```

Build for production

```bash
npm run build
```
